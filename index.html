keep.random

Make random service transparent and simple.
Based on Keep Random Beacon, we built Keep.Random for you. This is a powerful tool that you can use to get true randomness in a trustless way and apply it to your products!

Read the docs arrow_forward

Getting started

Step 1. Install the package 

npm install keep.random
Step 2. Use your favourite wallet to send tx. 

// import the module
import keepRandom from 'keep.random';

// send random transaction
const randomResult = await keepRandom.sendTx(web3);
Step 3. Get random result from keep.random. 

// get the result
if (randomResult) {
    console.log(randomResult);
} else {
    console.log('failed');
}
