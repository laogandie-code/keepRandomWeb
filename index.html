keep.random

Random service made simple.
keep.random is a random service powered by Keep Network, which you can get random on time without having to trust a third party.

Transparent and Traceable.
Read the docs arrow_forward

Why use it?

keep.random makes it easy to get random number everywhere. You don't need to trust any other parties, all random result can be traced on Ethereum.

Getting started

1. Install the client 

npm install keep-random
2. Use your favourite wallet to send tx. 

// import the module
import keepRandom from 'keep-random';

// send random transaction
const randomResult = await keepRandom.sendTx(web3);
3. Get random result from keep.random. 

// get the result
console.log(randomResult);
